01. 개요

프로젝트 목표

AI 기반 CCTV 화재 감지 및 119 신고 지원 시스템을 통해 관제 인력의 대응 시간을 단축하고 화재 초기 대응 역량을 향상한다.

02. 기존 문제점

화재 상황은 단순 CCTV 확인만으로는 신속한 판단과 대응이 어렵습니다.

CCTV는 많지만 사람이 항상 볼 수 없다

CCTV는 영상을 촬영할 뿐 상황을 스스로 판단하지 못합니다. 관리자가 자리를 비우거나 여러 화면을 동시에 봐야 하는 상황에서는 위험 상황을 놓칠 수 있으므로, 일정 위험도 이상이면 자동모드로 119 신고까지 이어지는 구조가 필요합니다.

자동모드 기반 무인 대응 필요

화재 발생 시 상황 판단이 늦다

화재 여부, 발생 위치, 사람 존재 여부, 화재 규모를 관리자가 직접 확인해야 합니다. 이 과정에서 초기 대응 시간이 지연될 수 있습니다.

초기 대응 시간 지연

단순 탐지만으로는 대응이 어렵다

YOLO등의 모델을 통해 화염, 연기, 사람을 탐지할 수 있지만 위험도와 신고 필요 여부까지 판단하지는 못합니다. 객체 탐지 결과를 상황 판단으로 연결하는 과정이 필요합니다.

객체 탐지와 의사결정은 다름

반복 탐지 시 중복 알림이 발생할 수 있다

일반적인 화재 상황에서는 같은 CCTV에서 연기와 화염이 반복적으로 탐지됩니다. 이벤트 단위로 묶지 않으면 동일 상황에 대한 알림과 분석 요청이 과도하게 발생할 수 있습니다.

이벤트 관리와 알림 제어 필요

MVP 목표

YOLO 기반 화재 후보 감지 → Issue 기반 중복 제어 → VLM 기반 상황 해석 → 119 신고 문구 생성을 통해 관리자의 의사결정을 지원합니다.
